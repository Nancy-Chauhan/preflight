# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-25

### Added
- Initial release of the preflight skill
- Service detection for 40+ services across 11 categories (AI/LLM, Database, Email, Hosting, Auth, Payments, Storage, Analytics, Cache/Queue, Messaging)
- Flow tracing and multiplier detection
- Operational simulation with daily narrative
- Bottleneck and breaking point analysis with PASS/WARN/FAIL severity
- Infrastructure dependency mapping with change impact tables
- Full tier pricing data (not just free tier)
- Example outputs for 3 scenarios (small SaaS, self-hosted, map-only)
